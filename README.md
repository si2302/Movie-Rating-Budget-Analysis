# Project Description:
This project presents an interactive Tableau dashboard that analyzes movie ratings across different genres and years. The dashboard provides insights into rating trends, audience preferences, and the relationship between critic and audience scores.

Movie Rating Trend Over Time
Analyzes how Rotten Tomatoes ratings have changed across different release years.

Distribution of Ratings by Genre
Visualizes how audience ratings are distributed across various movie genres.

Genre-wise Movie Share (Pie Chart)
Shows the proportion of movies in each genre category.

Average Rating by Genre
Compares the average Rotten Tomatoes ratings across genres to identify top-performing categories.

Correlation Analysis (Scatter Plot)
Examines the relationship between Rotten Tomatoes ratings and Audience ratings to identify patterns and trends.
